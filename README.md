# Hostel Complaint Management System

## 📌 Description

A web-based system for managing hostel complaints with role-based access.

## 👨‍🎓 Features

* Student login/register
* Submit complaints with priority
* View complaint status
* Admin panel to manage complaints
* Status updates (Pending → In Progress → Resolved)
* Analytics dashboard

## 🛠️ Technologies Used

* HTML, CSS
* PHP
* MySQL

## 🔐 Security

* Password hashing using `password_hash()`
* Session-based authentication

## 📊 Modules

* Login System
* Complaint System
* Admin Panel
* Analytics Dashboard

## 🚀 How to Run

1. Start PHP server:

   php -S localhost:8000
   
2. Open:

   http://localhost:8000
